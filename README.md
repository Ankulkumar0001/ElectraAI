# ⚡ ElectraAI: Diagnostics & Repair System

ElectraAI ek advanced AI-powered web application hai jo users ko unke faulty electronics devices ko diagnose karne, repair steps samajhne aur troubleshooting insights paane me madad karta hai. Yeh system deep image diagnostics (AI Vision) aur details-based manual forms dono ko support karta hai taaki electronic faults ka exact analysis lagaya ja sake.

## ✨ Key Features & Technical Capabilities

- **👁️ AI Vision (Quick Image Diagnostics):** Users can upload a photo of a faulty device, damaged PCB, or error screen for instant structural and visual fault detection.
- **📷 Live Camera Integration:** Features an in-app webcam capture capability to quickly frame and record/scan hardware damage in real-time.
- **📝 Multi-Parameter Diagnostic Form:** Supports exhaustive technical inputs including:
  - **Brand & Model Identification:** (e.g., HP, Samsung, LG, Galaxy Series)
  - **Temporal Context:** Purchase Year tracking to evaluate device deprecation.
  - **Error Code Extraction:** Field targeting direct software/hardware flags (e.g., `0x0000007B`, `PE Error`).
  - **Pre-set Symptom Checkboxes:** Fast selection for Screen damage, Battery/Overheating, Boot loops, Speaker muffling, and Water damage.
  - **Voice-to-Text Issue Description:** Integrated mic input allowing users to speak out symptoms comfortably.
- **🤖 Contextual ElectraAI Chat Assistant:** Provides immediate multi-lingual (English/Hindi) step-by-step resolution logs, safety protocols, and suggested action maps directly mapped to the scanned system.
- **🔑 Client-Side Groq API Validation:** To maximize resource scaling and maintain key privacy, users input their own Groq API keys which process lightning-fast inference on their local system.

## 🛠️ Tech Stack

- **Frontend Core:** HTML5, Modern JavaScript (ES6+), CSS3
- **Styling Architecture:** Tailwind CSS / Bootstrap framework for responsive fluid design cards
- **Core Engine & LLM Processing:** Groq API Cloud Client (Powering real-time Vision & Chat inferences)
- **Deployment Platform:** Netlify Cloud Hosting
